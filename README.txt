En la clase Mundial2018 se encuentra toda la interaccion con el usuario, los switch con los menu de opciones y la carga inicial desde el archivo de texto.
En la clase TablaEquipos se encuentran todos los metodos que interactuan con la estructura Diccionario ( equipos ).
En la clase MapaCiudades se encuentran todos los metodos que interactuan con la estructura Grafo (ciudades, viajes).
En la clase PartidosHashMap se encuentran todos los metodos que interactuan con la estructura HashMap de java ( partidos).
La clase KeyPartido es la que genera la clave para guardar en el HashMap, donde su constructor deja el eq1 como el menor alfabeticamente entre los 2.
